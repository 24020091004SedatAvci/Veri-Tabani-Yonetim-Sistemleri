# Sepet Uygulaması - Java Edition

Java dilinin gücüyle, JDBC (Java Database Connectivity) kullanılarak geliştirilmiş masaüstü uygulamasıdır.

### ✨ Özellikler
* SQL Server JDBC Driver entegrasyonu.
* OOP (Nesne Yönelimli Programlama) prensiplerine uygun kod yapısı.
* Temiz ve işlevsel kullanıcı arayüzü.
