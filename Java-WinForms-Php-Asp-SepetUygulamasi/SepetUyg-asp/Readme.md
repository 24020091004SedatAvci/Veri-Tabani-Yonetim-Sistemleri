# Sepet Uygulaması - ASP.NET Web Forms

WinForms mantığının web ortamına taşındığı, olay tabanlı (event-driven) geliştirilmiş profesyonel web uygulamasıdır.

### ✨ Özellikler
* **GridView** kontrolleri ile gelişmiş veri listeleme.
* **Postback** mekanizması ile sunucu taraflı işlem yönetimi.
* `Web.config` üzerinden merkezi bağlantı yönetimi.
* Satır tıklama özelliği (RowDataBound) ile modern kullanıcı deneyimi.
