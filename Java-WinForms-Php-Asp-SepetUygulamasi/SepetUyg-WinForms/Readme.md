# Sepet Uygulaması - WinForms Edition

Sistemin temelini oluşturan, ADO.NET mimarisi kullanılarak geliştirilmiş masaüstü uygulamasıdır.

### ✨ Özellikler
* Müşteri ekleme ve durum yönetimi (Aktif/İptal/Tamamlandı).
* Seçilen müşteriye özel ürün sepeti oluşturma.
* SQL Server ile senkronize veri yönetimi.
