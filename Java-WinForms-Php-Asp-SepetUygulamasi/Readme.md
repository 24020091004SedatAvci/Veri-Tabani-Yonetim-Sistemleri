# Multi-Platform Sepet Uygulaması 

Bu proje, bir müşteri yönetim ve sepet sisteminin farklı teknolojiler (Desktop & Web) kullanılarak nasıl inşa edileceğini gösteren bir "Cross-Platform" çalışmasıdır. Tüm uygulamalar merkezi bir **MS SQL Server** veritabanına bağlıdır.

## 🛠 Kullanılan Teknolojiler
* **Database:** Microsoft SQL Server
* **Desktop:** C# WinForms, Java (Swing/JavaFX)
* **Web:** PHP 8.x, ASP.NET Web Forms (.NET Framework 4.8)

## 📂 Klasör Yapısı
* `SepetUyg-WinForms`: Masaüstü C# uygulaması.
* `SepetUyg-php`: PHP ve SQLSRV sürücüsü ile geliştirilmiş web sürümü.
* `SepetUyg-asp`: ASP.NET Web Forms mimarisi ile geliştirilmiş web sürümü.
* `SepetUyg-java`: Java tabanlı masaüstü çözümü.
