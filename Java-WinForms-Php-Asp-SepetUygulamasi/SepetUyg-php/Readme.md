# Sepet Uygulaması - PHP Edition

WampServer ortamında geliştirilmiş, PHP ve SQL Server (sqlsrv) entegrasyonuna sahip web çözümüdür.

### ✨ Özellikler
* Tamamen responsive (duyarlı) mor tema tasarımı.
* URL parametreleri ile müşteri tabanlı sepet filtreleme.
* PHP `sqlsrv` kütüphanesi ile yüksek performanslı veritabanı iletişimi.

![PHP Web Arayüzü](screenshots/php_web.png)

### 🚀 Kurulum
1. `db.php` dosyasındaki SQL Server bağlantı bilgilerini güncelleyin.
2. `sqlsrv` eklentisinin PHP'de aktif olduğundan emin olun.
3. Projeyi `www/` klasörüne taşıyıp tarayıcıdan erişin.
